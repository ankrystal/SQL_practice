Q. Write an SQL query to report the nth highest salary from the Employee table. <br>

```mysql
CREATE FUNCTION getNthHighestSalary(N INT) RETURNS INT
BEGIN
SET N = N-1;
  RETURN (
        SELECT DISTINCT Salary FROM Employee ORDER BY Salary DESC 
        LIMIT 1 OFFSET N
  );
END
```
